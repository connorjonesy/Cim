# Cim
Welcome to Cim (Connor's Vim)

Have you ever accidentally typed
 `cim file.txt` 
 instead of 
 `vim file.txt`?

Yeah. Me too. 

Well. Now it's no problem! You'll open up a terminal UI text editor just like Vim!

The only difference is Cim is way worse.

But hey, atleast you can edit your file!

## About Cim
Cim is a terminal UI text editor. I am working on this project to learn about the Go programming language and Bubble Tea (a Go framework for building terminal apps).
